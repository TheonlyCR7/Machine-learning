#### `%timeit` 测试运行时间



`%timeit L = [i**2 for i in range(1000)]`  测试一行代码的运行时间

![image-20200603202945777](img/image-20200603202945777.png)

![image-20200603203112974](img/image-20200603203112974.png)



```
%%timeit 
L = []
for n in range(1000):
    L.append(n ** 2)
```
